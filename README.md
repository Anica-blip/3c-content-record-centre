## 🗃️ What Is the Content Record Centre?

The Content Record Centre is a private workflow tool — not a public-facing app.
It is where every piece of 3C content gets registered, archived, and tracked from creation through to public distribution.

**Inside the Record Centre:**
- 📇 **Index List** — searchable record of every registered piece of content, filterable by category, persona, title, platform, month/year, or number
- 🪪 **Three-Card System** — Card 1 (front label), Card 2 (production/content panel), Card 3 (distribution panel, per platform)
- 🔢 **Content ID Numbering** — a structured sequence (platform + format + persona + month + year + number) that becomes the glue holding the ecosystem together
- 📤 **PDF Export** — every card can be downloaded for offline work
- ☁️ **Cloudflare R2 + Worker** — no traditional database; records are stored as JSON files
